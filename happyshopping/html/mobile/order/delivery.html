  • 택배
    택배 진행 순서
    1. 1상품 주문 및 결제
      상세
      당일발송 가능 결제시간 ※ 조립PC의 경우
      평일 : 오전 10:00~오후 4:00
      토요일 : 오전 10:00~오후 4:00
      평일 : 오전 10:00~오후 4:00 토요일 : 오전 10:00~오후 11:00
    2. 2상품 준비 및 포장
      상세
      부품일 경우 소요시간
      결제 확인 후 2시간 소요
      조립PC의 경우 소요시간
      결제 확인 후 3시간 소요
      결제 확인 후 상품 준비 및 포장
    3. 3택배 발송

      발송 후 송장번호 문자메시지가 발송 됩니다.

    4. 4배송 완료

      고객님이 선택한 배송지로 안전하게 배송됩니다.

    이용안내 택배추적은 제품 발송일 다음날 오전부터 택배사 홈페이지에서 확인 가능 합니다.
    배송기간은 제품 발송 후 1~2일 이내에 받으실 수 있습니다.

      토요일 : 오전 10:00~오후 4:00
      평일 : 오전 10:00~오후 4:00 토요일 : 오전 10:00~오후 11:00
    2. 2상품 준비 및 포장
      상세
      부품일 경우 소요시간
      결제 확인 후 4시간 소요
      조립PC의 경우 소요시간

      결제 확인 후 상품 준비 및 포장
    3. 3택배 발송

      퀵서비스 발송완료 후 문자메시지가 발송됩니다.

    4. 4배송 완료

      고객님이 선택한 배송지로 안전하게 배송됩니다.

    이용안내 배송 지연 상품 / 대량구매상품 / 조립PC 수량 주문은 준비시간이 지연될 수 있으니 담당자와 통화하여 상품 준비 소요시간을 꼭 확인하시기 바랍니다.
    입금마감시간 이후에 결제확인이 된 주문건은 익일 수령이 가능합니다. (단, 주말/공휴일 제외)

      상세

      다음날 발송됩됩니다.
    2. 2상품 준비 및 포장
      상세

      결제 확인 후 상품 준비 및 포장
    3. 3경동화물 발송

      경동화물 상품 발송 후 송장번호 문자메시지가 발송됩니다.

    4. 4배송 완료

      고객님이 선택한 영업소로 방문하셔서 직접 상품 수령하시면 됩니다.

    이용안내 경동화물은 고객님께서 선택하신 영업소로 방문하여 상품을 수령해야 합니다.
