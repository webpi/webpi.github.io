공통 사항
  • - 선불통화권 잔액이 모두 소진된 경우 발신통화기능이 정지되며, 수신은 발신통화 정지일로부터 10일간만 가능합니다.
  • - 잔액소진 후 수신가능기간은 15일이며, 잔액소진 상태에서 90일 경과 시 직권해지 될 수 있으니 유의하시기 바랍니다.
  • - 부가서비스는 이동통신사(LGT)와 동일합니다. 단, 무선인터넷은 아래 선불 데이터 상품을 미리 구매/충전하여 사용 가능합니다.
과금정보
상품명 이용요금 데이터량 사용기간
선불 데이터 100MB 5,000원 100MB 30일
선불 데이터 500MB 8,000원 500MB 30일
선불 데이터 1GB 12,000원 1GB 30일
  • - 데이터서비스는 이용요금을 미리 구매/충전하여야 사용가능합니다.
  • - 데이터량 소진시 무선데이터가 자동차단되며, 재충전하여 사용가능합니다.
  • - 단, 데이터량 소진시 30일 이내, 동일상품 재충전 불가하며, 다른 상품으로만 충전 가능
    • 예) 5월 1일 100MB 데이터 서비스 충전 후 100MB 소진 시, 30일 이내 동일상품(100MB) 재충전 불가
    • 예) 5월 1일 100MB 데이터 서비스 충전 후 100MB 소진 시, 30일 이내 다른상품(500MB, 1GB) 충전 가능
  • - 사용기간이 지나면 남은 데이터량은 모두 자동 소진됩니다.
  • - 서비스 신청 후 중도 변경/해지시 이미 결제된 금액은 환불되지 않습니다.
  • - 무통장입금(가상)계좌로 입금(충전) 시 사용기간은 입금(충전)시점으로 부터 30일간 입니다.
  • - MB Tip : 통상 포털 초기 페이지 용량은 0.5~1MB, MP3 벨소리 하나 3~5MB, 동영상 1분에 2~3MB 정도 소요됩니다.
  • - 무선인터넷 1MB는 1,024KB 입니다.
