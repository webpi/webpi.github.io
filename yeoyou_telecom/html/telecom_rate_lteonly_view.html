메세지
  • - 메시지 사용건수가 1일 200건 이상, 월 10회 초과 시
  • - 메시지 수신처가 3,000명 초과 시
  • - 시지 사용건수가 일 500건 초과 시
  • - 기타 이용약관 규정 위반 시(물리적 장치 등을 통해 발송, 제3자 임대 등)
데이터
  • - 음성무한 데이터 50.9 요금제 LTE비디오포털 이용 예시 : 전용 데이터 3.3GB → 요금제 기본 제공 데이터 6.6GB중 데이터 사용 후 잔여량 → 종량 과금(20.48원/MB(22.53원/MB))
